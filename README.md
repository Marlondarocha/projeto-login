# Projeto Login - Curso em Vídeo 📱

Projeto desenvolvido durante o curso de HTML5 e CSS3 do **CURSO EM VÍDEO - Gustavo Guanabara**. Objetivo focado em conseguir criar uma página fundamentada em **responsividade** e uma grande utilização de **estilização**.

## 📚 Tecnologias e conceitos aplicados:
**CSS**: aplicação organizada de estilos básicos e design visual. \
**Design responsivo**: utilização de Media Queries em diferentes telas. \
**Layout**: centralização de elementos e o emprego de espaçamentos.
